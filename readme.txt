Simple AdTaily Widget for WordPress

== Description ==

Super simple widget for AdTaily ads. Shows ads in your widgetized sidebar. You must setup your website on AdTaily.pl first.
  
== Installation ==

1. Upload `adtaily-widget-light` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Setup it in `/wp-admin/widgets.php` (drag adtaily-widget-light to widget area)
